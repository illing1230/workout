# Changelog

Health App의 모든 주요 변경사항이 이 파일에 기록됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 기반으로 하며,

## [1.0.0] - 2025-11-20

### ✨ Added (새로운 기능)

#### 🏠 홈 화면
- 오늘의 목표 대시보드 구현
  - 운동 횟수 추적
  - 운동 시간 추적 (분)
  - 칼로리 소모량 추적
- 그래디언트 헤더 디자인
- 주간 통계 요약 카드
- 빠른 운동 시작 버튼

#### 💪 운동 라이브러리
- 10가지 운동 종류 추가
  - 런닝 (10 kcal/분)
  - 벤치프레스 (7 kcal/분)
  - 스쿼트 (8 kcal/분)
  - 데드리프트 (9 kcal/분)
  - 요가 (4 kcal/분)
  - 사이클 (8 kcal/분)
  - 풀업 (8 kcal/분)
  - 플랭크 (5 kcal/분)
  - 버피 (12 kcal/분)
  - 숄더프레스 (6 kcal/분)
- 카테고리 필터링 시스템 (9개 카테고리)
- 검색 기능
- 난이도 표시 (초급/중급/고급)
- 운동 상세 정보 모달
- 주요 근육 표시
- 칼로리/분 정보

#### 📋 운동 프로그램
- 5가지 체계적인 프로그램 구현
  - 🌱 초보자 프로그램 (4주, 주 3일)
  - 💪 근력 강화 프로그램 (8주, 주 4일)
  - 🔥 체지방 감량 프로그램 (6주, 주 5일)
  - 🧘 유연성 향상 프로그램 (4주, 주 4일)
  - ⚡ 풀바디 챌린지 (12주, 주 6일)
- 프로그램별 그래디언트 카드 디자인
- 상세 루틴 모달
- 프로그램 시작 기능
- 일일 운동 계획 표시

#### 📅 운동 기록
- React Native Calendars 통합
- 캘린더 기반 운동 추적
- 운동 추가/수정/삭제 기능
- 날짜별 운동 목록
- 운동 타입별 아이콘
- 칼로리 자동 계산
- 메모 기능
- 최근 5개 운동 표시

#### 💬 AI 챗봇
- 대화형 채팅 인터페이스
- 지능형 응답 시스템
  - 운동 추천
  - 식단 조언
  - 통계 분석 (실제 데이터 연동)
  - 동기부여 메시지
  - 부상/건강 관리 조언
  - 프로그램 상담
- 제안 칩 (Suggestion Chips)
- 빠른 액션 버튼 (4개)
- 타이핑 인디케이터 애니메이션
- 자동 스크롤
- 키보드 대응

#### 🎨 디자인 시스템
- Burnfit 스타일 다크 테마 구현
- 색상 팔레트
  - Primary: #3292FF (Burnfit Blue)
  - Secondary: #FF6B6B (Coral Red)
  - 다크 배경: #0F1419
  - 표면: #242D34
- 7가지 그래디언트 조합
- 통일된 간격 시스템
- 타이포그래피 시스템
- 그림자 시스템

#### 🔧 기술 구현
- Context API 기반 상태 관리
  - WorkoutContext 구현
  - 전역 운동 데이터 관리
  - 목표 설정 관리
- AsyncStorage 로컬 저장
  - 운동 데이터 영구 저장
  - 목표 설정 저장
  - 앱 재시작 시 자동 로드
- React Navigation 설정
  - Bottom Tab Navigator (5개 탭)
  - 탭 아이콘 애니메이션
  - 다크 테마 탭 바
- 커스텀 컴포넌트
  - LinearGradient (웹 호환)
  - 모달 컴포넌트들
